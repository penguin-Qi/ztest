ztest

A modern, feature-rich testing framework for C++ development that helps you write better tests, faster.

Comprehensive Testing

Support for unit testing, benchmarking, parameterized tests, and performance analysis in one unified framework.

Modern UI

Intuitive graphical interface with real-time resource monitoring, detailed test visualization, and dark/light theme support.

Powerful Features

Parallel execution, test filtering/sorting, setup/teardown hooks, and comprehensive reporting (HTML, JSON, JUnit).

Extensible Architecture

Modular design with plugin system, LRU caching for test data, and customizable test runners.

Benchmark Tools

Integrated benchmarking with detailed performance metrics and comparison capabilities.

Test Management

Robust test registry, result management, and data-driven testing capabilities.
